## GRAMMAR EXPLANATION - WEEK 4

### "THERE IS" and "THERE ARE"

#### Rule 1: BASIC STRUCTURE

"**There is**" and "**There are**" describe WHAT exists or WHERE things are.

```
STRUCTURE:
  There + is/are + (a/some) + noun + location

There is     + singular noun       (1 object)
There are    + plural noun         (2+ objects)

EXAMPLES:
  ✅ "There **is** a bed in the bedroom."     (1 bed = singular)
  ✅ "There **are** chairs in the dining room." (2+ chairs = plural)
  ❌ "There is chairs"                        (wrong = plural)
  ❌ "There are a chair"                      (wrong = singular)
```

#### Rule 2: AFFIRMATIVE vs. NEGATIVE

```
AFFIRMATIVE:
  There is a sofa in the living room.
  There are pictures on the wall.

NEGATIVE:
  There **isn't** a bed in the living room.     (is not)
  There **aren't** any chairs in the kitchen.   (are not)

QUESTIONS:
  **Is there** a table in the dining room?      → Yes, there is.
  **Are there** any windows in the bedroom?     → Yes, there are.
```

#### Rule 3: WORD ORDER

```
There + is/are + (a/some) + noun + preposition + location

Where is/are?    ↓
                There is one bed    in the bedroom
                       ↓              ↓        ↓
                    Object        Preposition  Place

Example: "There are three chairs around the table."
         There + are + three chairs + around + the table
         (location start → objects → where)
```

#### Example Sentences

```
1. "There **is** a refrigerator **in the kitchen**."
   → 1 refrigerator (singular) → "is"
   
2. "There **are** chairs **around the table**."
   → More than 1 chair (plural) → "are"
   
3. "There **is** a mirror **on the bathroom wall**."
   → 1 mirror (singular) → "is"
   
4. "There **isn't** a bed **in the living room**."
   → Negative: no bed in living room
   
5. "There **are** pictures **on the walls**."
   → Multiple pictures (plural) → "are"
```

#### Common Student Errors

| Error | Why | Correct | Explanation |
|-------|-----|---------|-------------|
| "There is chairs" | Confusing singular/plural | **There are** chairs | Plural nouns use "are" |
| "There are a table" | Wrong article + form | **There is a** table | Singular uses "is" |
| "Is there a sofa?" [No response pattern] | Not understanding question structure | Yes, there is. / No, there isn't. | Respond with "there is/are" |
| "There are one bed" | Not counting correctly | **There is one bed** | "One" = singular → use "is" |
