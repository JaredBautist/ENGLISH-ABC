## GRAMMAR EXPLANATION - WEEK 7

### "WOULD LIKE" vs. "LIKE"

#### Rule 1: "WOULD LIKE" (More Polite/Formal)

Used to express DESIRES in present moment, especially when ordering.

```
STRUCTURE:
  Subject + would + like + (a/some/the) + noun

CONJUGATION:
  I would like...
  You would like...
  He/She would like...
  We would like...
  They would like...

NOTE: "would" is the same for ALL subjects!
  (This is different from "do/does")

CONTRACTIONS:
  I'd like = I would like
  He'd like = He would like
  She'd like = She would like
  They'd like = They would like

EXAMPLES:
  "I would like a coffee."
  "She'd like some water."
  "Would you like rice?"
```

#### Rule 2: "LIKE" (General Preference)

Used for habits, general preferences, NOT in the moment.

```
STRUCTURE:
  Subject + like/likes + -ing / noun

EXAMPLES:
  "I like coffee." (general preference)
  "She likes reading." (habit)
  "They like fish." (preference)

COMPARISON:
  ‚úÖ "I like coffee" (in general, always)
  ‚úÖ "I would like a coffee" (right now, specific moment)
  
  ‚úÖ "Would you like tea?" (offering now)
  ‚úÖ "Do you like tea?" (general question)
```

#### Rule 3: QUANTIFIERS

```
STRUCTURE:
  Subject + like/would like + quantifier + noun

QUANTIFIERS:
  a little    = small amount (uncountable)
  some        = unspecified amount
  a lot of    = large amount
  a few       = small number (countable)

EXAMPLES:
  "I would like **a little** milk."     (small amount of milk)
  "I would like **some** rice."         (rice, unspecified)
  "She likes **a lot of** vegetables."  (many vegetables)
  "I would like **a few** carrots."     (a small number of carrots)

REMEMBER: 
  "a little" = uncountable (milk, water, butter)
  "a few" = countable (carrots, apples, eggs)
```
