## GRAMMAR EXPLANATION - WEEK 5

### LIKE + -ING FORM

#### Rule 1: THE "-ING" FORM (GERUND)

When using "like," the verb becomes "-ing" (called a gerund).

```
STRUCTURE:
  Subject + like/likes + verb-ing

BASE VERB → ADD "-ING"
  ✅ run → running
  ✅ swim → swimming
  ✅ cook → cooking
  ✅ dance → dancing
  ✅ read → reading

SPELLING CHANGES:
  Most verbs: just add "-ing"
    read → reading
    walk → walking
    
  Verb ends in 'e': drop 'e', add "-ing"
    make → ma-k-ing (drop 'e' first!)
    take → taking
    bake → baking
    
  Verb ends in consonant+vowel+consonant:
    sit → sitting (double the final consonant!)
    run → running
    swim → swimming
```

#### Rule 2: LIKE/LIKES WITH -ING

```
AFFIRMATIVE:
  I         like      running
  You       like      cooking  
  He/She    likes     reading     (add -s!)
  We        like      swimming
  They      like      dancing

NEGATIVE:
  I don't like running.
  She doesn't like dancing.

QUESTIONS:
  Do you like reading?
  Does he like playing soccer?
```

#### Rule 3: FREQUENCY ADVERBS PLACEMENT

```
Frequency adverbs usually go in the MIDDLE of sentences:

Position:
  Subject + FREQUENCY ADVERB + verb

Examples:
  I **always** exercise.
  She **usually** reads.
  He **sometimes** plays soccer.
  They **never** watch TV.

COMMON PATTERN:
  I always like + verb-ing
  She usually likes painting.
  He sometimes likes swimming.
  They never like dancing.
```

#### Example Sentences

1. "I **like running** in the morning."
2. "She **likes swimming** every summer."
3. "They **don't like** cooking."
4. "He **always likes** playing soccer."
5. "I **usually like** reading before bed."
6. "She **sometimes likes** dancing at parties."
7. "They **never like** watching TV."

#### Common Student Errors

| Error | Why | Correct | Explanation |
|-------|-----|---------|-------------|
| "I like to run" | Using infinitive | **I like running** | Use -ing form, not "to..." |
| "She like reading" | No -s with "likes" | **She likes reading** | He/she = "likes" (always add -s) |
| "I always like singing" | Frequency placement | **I like singing always** | Put frequency in middle |
| "run" (not "running") | Forgetting -ing | **running** | Always use -ing after like |
| "He don't like soccer" | Wrong auxiliary | **He doesn't like soccer** | He = "doesn't" |

*"Hello! My name is Carlos. I'm very active. I love running! I run every morning. I usually play soccer with friends on weekends. Sometimes I like cycling in the park. But I never like reading—it's boring for me!"*

   - A) Never | **B) Usually** | C) Always
