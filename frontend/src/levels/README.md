# Levels structure

- `a1-1/pages`: páginas y diapositivas del nivel A1.1
- `a1-1/components`: componentes usados por A1.1 (wrappers para componentes compartidos)
- `a1-2/pages`: páginas y diapositivas del nivel A1.2
- `a1-2/components`: componentes usados por A1.2 (wrappers para componentes compartidos)

Esta estructura permite separar por nivel sin duplicar lógica.
